# Spike Plan

## Inputs

- Unknowns and risks.
- Timebox.
- Required outputs.

## Outputs

- `docs/05-prds/<slug>_spike-plan.md`.

## Steps

1. List questions to answer.
2. Define timebox and approach.
3. Set exit criteria and deliverables.

## Verification

- Spike has explicit questions, timebox, and exit criteria.
