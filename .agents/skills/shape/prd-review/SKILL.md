# PRD Review

## Inputs

- PRD document.
- Acceptance criteria.
- Constraints and risks.

## Outputs

- Review notes and gaps list.

## Steps

1. Check goals, scope, and non-goals.
2. Verify acceptance criteria and verification plan.
3. Flag missing risks or unclear scope.

## Verification

- Review notes include any missing criteria, risks, or scope gaps.
