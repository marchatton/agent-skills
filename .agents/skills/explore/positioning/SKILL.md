# Positioning

## Inputs

- Target segment.
- Competitor set.
- Key differentiators.

## Outputs

- Positioning statement.
- Differentiation bullets.

## Steps

1. State target user and category.
2. Define differentiation and proof points.
3. Draft a concise positioning statement.

## Verification

- Positioning is specific to target segment and differentiators.
