# Customer Segmentation

## Inputs

- Problem area and market.
- Existing user data or hypotheses.
- Constraints and access channels.

## Outputs

- Segment list with defining traits.
- Primary target segment and rationale.

## Steps

1. List candidate segments and differentiators.
2. Score segments on need, willingness, and reach.
3. Pick the primary segment and note tradeoffs.

## Verification

- Primary segment is explicit with evidence and tradeoffs.
