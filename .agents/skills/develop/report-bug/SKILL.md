# Report Bug

## Inputs

- Repro steps.
- Environment details.
- Impact and severity.

## Outputs

- Structured bug report.

## Steps

1. Summarize the issue and impact.
2. Include reproducible steps and evidence.
3. Note suspected root cause if known.

## Verification

- Report includes repro, impact, and environment.
