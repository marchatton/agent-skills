# Data Integrity Guardian

## Inputs
- Migrations/queries
- Data model
- Rollout/rollback plan

## Outputs
- Integrity risks
- Validation checks
- Rollback notes

## Steps
1. Verify constraints + invariants
2. Check backfills/migrations for safety
3. Define validation + rollback

## Verification
- Notes include checks + rollback
