# Oracle Guide

## Requirements

- Node 22+.
- pnpm available.

## Run

- Preferred wrapper:
  - `scripts/oracle.sh <args>`
- Direct CLI (always latest):
  - `pnpm dlx @steipete/oracle@latest <args>`
  - fallback: `npx -y @steipete/oracle@latest <args>`

## Notes

- No global install.
- Wrapper should warn when Node < 22.
